# 🧠 Neurobridge Screener

### Cognitive & Learning Profile Assessment System

---

# Project Title

**Neurobridge Screener — AI-Driven Cognitive Learning Assessment Platform**

A structured questionnaire-based system that evaluates user learning behavior, cognitive traits, and study preferences, then generates an adaptive learning profile.

---

# Problem Statement

Students and learners often face:

* Difficulty understanding their learning style
* Lack of structured cognitive assessment tools
* No personalized learning insights
* No progress tracking for behavioral patterns
* Disconnected learning optimization tools
* No simple screener before adaptive learning systems

Most tools jump directly to content delivery without **understanding the learner first**.

There is a need for a **cognitive screener** that evaluates users before building learning systems.

---

# Solution

Neurobridge Screener provides a **guided questionnaire-based assessment** that:

* Evaluates attention span
* Identifies learning style
* Measures reading difficulty
* Tracks behavioral responses
* Generates cognitive profile
* Displays structured results

The platform acts as a **foundation layer** for adaptive learning systems.

# How to Run

## 1. Clone Repository

```bash
git clone https://github.com/pareesa969/Neurobridge-Screener.git
```

## 2. Navigate to Project

```bash
cd neurobridge-screener
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Run Development Server

```bash
npm run dev
```

## 5. Build Production

```bash
npm run build
```
